    K-Means

    1. 先在座標中隨機標上K個中心點
    2. 對於每一個index塗上離該index最靠近的中心點顏色
    3. 將中心點移動到所有該中心點顏色的index(平均)
    4. 重複2~3步驟

    註：距離採用歐式距離

    K-Nearest Neighbors

    1. 對於每一個尚未標記的點，找到離該點最近的k個點，將顏色塗為占比最多的顏色。

    SVM

    對於資料經過維度轉換後使其可以分割

    將一個視為1，其他視為-1

    使用多個hyperplane來分類 看每個大於0或是小於0

    若位於兩區塊交集，選擇較遠的，例如圖二中間，X距離紅色較遠，分類為紅色

    若位於三者交集，選最近，例如圖二右邊，分類為綠色

    ```python= import os import pickle from datetime import datetime class MLPlay: def init (self, ai_name, args, *kwargs): print(ai_name) self.scene_info = [] self.commands = [] self.data = {"scene_info": [], "command": []}

        self.ball_served = False
    self.previous_ball = (0, 0)
    self.pred = 100 #預設板子x位置為100
    self.platform_y = 400 #板子y位置
    self.ball_speed_y = 7 #每frame球在y軸方向移動速度為+-7
    self.platform_width = 200 #板寬


def update(self, scene_info, *args, **kwargs):
    #Make the caller to invokereset() for the next round.
    if (scene_info["status"] == "GAME_OVER" or
            scene_info["status"] == "GAME_PASS"):
        return "RESET"
    if not self.ball_served:
        self.ball_served = True
        self.previous_ball =  scene_info["ball"]
        command = "SERVE_TO_RIGHT"
    else:
        # rule code
        self.pred = 100
        if self.previous_ball[1] - scene_info["ball"][1]> 0: #球正在往上;[0]=x座標[1]=y座標
            pass
        else:#球正在往下,判斷球的落點
            distance_platform_ball_y = self.platform_y =scene_info["ball"] [1] 
            ball_speed_x = scene_info["ball"][0] - self.previous_ball[0]
            self.pred = scene_info["ball"][0] + (distance_platform_ball_y // self.ball_speed_y) * ball_speed_x
        #若self.pred在0-200之外,運用反射原理去計算最後落在0-200之間的落點
        section = (self.pred // self.platform_width)
        if(section % 2 == 0):
            self.pred = abs(self.pred - self.platform_width * section)
        else:
            self.pred = self.platform_width- abs(self.pred - self.platform_width * section)
        #板寬40,+20為中心點,+-5為板子每frame移動速度
        if scene_info["platform"][0]+20 +5 < self.pred:
            command = "MOVE_RIGHT"
        elif scene_info["platform"][0]+20 - 5 > self.pred:
            command = "MOVE_LEFT"
        else:
            command = "NONE"
    self.scene_info.append(scene_info)
    self.commands.append(command)

    self.previous_ball = scene_info["ball"]
    return command

def reset (self):
    self.ball_served = False
    self.data["scene_info"] = self.scene_info
    self.data["command"] = self.commands
    filepath = "log/"
    if not os.path.exists(filepath):
        os.makedirs(filepath)
    with open(os.path.join(os.path.dirname(__file__), \
        '../log/scene_info_{:%Y_%m_%d_%H_%M_%S}.pickle'.format(datetime.now())),\
            'wb') as f:
        pickle.dump(self.data, f)
