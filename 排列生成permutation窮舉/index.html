    排列生成permutation¶

    資料來源

    有三個函數可使用

    支援陣列資料結構 ( 記憶體位置連續儲存 ) 如： - int v[N] - vector\<int> v

    is_permutation( )¶

    is_permutation( 陣列A開頭指針 , 陣列A結尾指針 , 陣列B );

    判斷陣列 B 是否為陣列 A 排序後的結果

    也可把它看成集合比對器(集合不看順序)，如下：

    bool is_permutation( 陣列A開頭指針 , 陣列A結尾指針 , 陣列B ){

    if A集合包含於B集合 {
      return true;
    }

    reurn false;

}

    舉例：¶

    回傳true： 程式碼： 
    #include <bits/stdc++.h>
using namespace std;

int main(){

    char c[]("123");
    char v[]("132");
    cout<<is_permutation(c,c+3,v);

}
    輸出： 
    1
    回傳false： 程式碼： 
    #include <bits/stdc++.h>
using namespace std;

int main(){

    char c[]("123");
    char v[]("133");
    cout<<is_permutation(c,c+3,v);

}
    輸出： 
    0
    不同長度，回傳true： 程式碼： 
    #include <bits/stdc++.h>
using namespace std;

int main(){
    char c[]("123");
    char v[]("1234");
    cout<<is_permutation(c,c+3,v);

}
    輸出： 
    1

    prev_permutation( )¶

    prev_permutation( 陣列開頭指針 , 陣列結尾指針 );

    找出前一個字典序排列，找到回傳true，沒找到回傳false

    程式碼 

    #include <bits/stdc++.h>
using namespace std;

int main(){
    char c[]("321");
    while(prev_permutation(c,c+3)){
        cout<<c<<"\n";
    }
}
    輸出： 
    312
231
213
132
123

    next_permutation( )¶

    next_permutation( 陣列開頭指針 , 陣列結尾指針 );

    找出前一個字典序排列，找到回傳true，沒找到回傳false

    程式碼 

    #include <bits/stdc++.h>
using namespace std;

int main(){
    char c[]("123");
    while(next_permutation(c,c+3)){
        cout<<c<<"\n";
    }
}
    輸出： 
    132
213
231
312
321

    製作窮舉¶

    程式碼： 

    #include <bits/stdc++.h>
using namespace std;
int main(){
    char c[]("12345");
    while(next_permutation(c,c+strlen(c))){
        cout<<c<<";";
    }
}
