    輸出結尾空白¶

    如果要輸出一段資料，但是結尾不需要中間分隔物，該如何做呢?

    眾所周知~ 在宣告陣列時char x[6]="12345";時，程式在之後遇到x就會回傳為"12345"

    程式碼： 

    #include <bits/stdc++.h>
using namespace std;
int main(){
    char x[6]="12345";
    cout<<x<<"\n";
    cout<<x[3];
}
    輸出： 
    12345
4

    所以也可以直接把x替代成"12345"，如下：

    程式碼： 

    #include <bits/stdc++.h>
using namespace std;
int main(){
    char x[6]="12345";
    cout<<"12345"<<"\n";
    cout<<"12345"[3];
}
    輸出： 
    12345
4

    那麼接下來我們就可以做有趣的事情了

    程式碼： 

    #include <bits/stdc++.h>
using namespace std;
int main(){
    char c[]("12345");
    for(int i=0;i<strlen(c);i++){
        cout<<c[i]<<"abcde" [i];
    }
}

    輸出： 

    1a2b3c4d5e

    這段程式碼在輸出英文時根據 i 輸出不同英文 換個方向，要放入布林值也是可以的

    程式碼： 

    #include <bits/stdc++.h>
using namespace std;
int main(){
    char c[]("12345");
    for(int i=0;i<strlen(c);i++){
        cout<<c[i]<<";" [i==strlen(c)-1];
    }
}

    輸出： 

    1;2;3;4;5
